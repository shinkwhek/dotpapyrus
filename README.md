## Installation

1. two best ways

```
$ bash -c "$(curl -fsSL https://git.io/dotpapyrus)"
```

or

```
$ bash -c "$(wget -qO - https://git.io/dotpapyrus)"
```

2. ```$ make install```


## Update

```
$ make update
```
