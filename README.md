## Installation

1. two best ways

```
$ bash -c "$(curl -fsSL https://raw.githubusercontent.com/shinkwhek/dotpapyrus/master/etc/install.sh)"
```

or

```
$ bash -c "$(wget -qO - https://raw.githubusercontent.com/shinkwhek/dotpapyrus/master/etc/install.sh)"
```

2.
```$ make install```


## Update

```
$ make update
```
